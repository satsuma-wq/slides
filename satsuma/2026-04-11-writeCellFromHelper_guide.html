writeCellFromHelper

スプレッドシートのドロップダウン制限をバイパスして、GAS から指定セルへ素早く値を書き込む常設ヘルパー関数です。

このツールでできること

物件管理表「物件紹介」シートには、ステータス列などにドロップダウン（拒否モード）が設定されています。このツールはその制限を自動で迂回し、Claude や GAS から任意の値を確実に書き込めるようにします。

  • ⚡
    バリデーションを自動クリア → 値を書き込み → バリデーションを元に戻す
  • 🔁
    使用後はパラメータセル（Z1・Z2）を自動クリア。痕跡なし
  • 📌
    常設関数なので毎回コードを書き直す必要なし
  • 🤖
    Claude に「○行目のステータスを△△にして」と指示するだけで自動実行も可能

使い方（3ステップ）

  1. 1
    Z1セルに書き込み先のセル番地を入力

    「物件紹介」シートの Z1 に F6 や D8 などを入力します。

  2. 2
    Z2セルに書き込む値を入力

    Z2 に 概要書作成済み などの値を入力します。

  3. 3
    GASエディタで関数を実行

    GAS エディタを開き、writeCellFromHelper を選択して「▶ 実行」をクリックします。

💡 Claudeと連携する場合 「D8に概要書作成済みを入れて」と Claude に伝えるだけで、Z1・Z2 の設定から GAS 実行まで自動で行います。

GASコード（AI連携ツール.gs に追加）

以下のコードを GAS プロジェクト「資料未請求アラート」の AI連携ツール.gs の末尾に貼り付けて保存してください。

AI連携ツール.gs 
// ========== セル書き込みユーティリティ ==========
// 使い方：物件紹介シートの Z1 に書き込み先セル番地（例: "F6"）、
//         Z2 に書き込む値（例: "概要書作成済み"）を入力してから実行する。
function writeCellFromHelper() {
  const ss = SpreadsheetApp.openById(
    '1YX74czZ6-vPT8NrxJL4_eNcul4xv9KLr3UqStVRKmgU'
  );
  const sheet = ss.getSheetByName('物件紹介');
  const cellAddr = sheet.getRange('Z1').getValue();
  const value    = sheet.getRange('Z2').getValue();

  if (!cellAddr) {
    Logger.log('Z1が空です。セル番地を入力してください。');
    return;
  }

  const range = sheet.getRange(cellAddr);
  const v = range.getDataValidation();  // バリデーション退避

  range.clearDataValidations();  // バリデーション一時解除
  range.setValue(value);           // 値を書き込む
  SpreadsheetApp.flush();          // 即時反映
  if (v) range.setDataValidation(v);  // バリデーション復元

  sheet.getRange('Z1:Z2').clearContent();  // パラメータをクリア
  Logger.log('書き込み完了: ' + cellAddr + ' = ' + value);
}
