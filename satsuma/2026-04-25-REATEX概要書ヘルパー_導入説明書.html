REATEX 概要書ヘルパー
導入説明書

対象者：社内メルマガ・LINE配信担当者
バージョン：v1.2.1
作成：株式会社リアテクス

✨このツールでできること

物件概要書ページを開いた状態で拡張機能ボタンを押すだけで、以下4種類の文書を自動生成します。

📧
メルマガ
投資家向けメルマガ用HTMLを自動生成（写真・KPIボックス付き）
💬
業者LINE
業者向けLINE紹介文を自動生成。短縮URL差し込みにも対応
📲
Lステップ
Lステップ配信テキストをワンクリックで自動生成
🤖
AIチャット
Claude AIに修正指示を出して文章をリアルタイム編集

🚀導入手順

1

ZIPファイルをダウンロード・解凍する

以下のGoogle Driveフォルダから ZIP ファイルをダウンロードしてください。

Google Drive でZIPファイルを開く

ダウンロードした ZIP ファイルをダブルクリックして解凍します。

reatex-helper-v1.2.1.zip

解凍すると reatex-helper-v1.2.1 フォルダが作成されます。

⚠️ このフォルダは削除しないでください。拡張機能として使い続ける限り必要です。
2

Chromeに拡張機能をインストールする

  1. Chromeのアドレスバーに chrome://extensions/ を入力してEnter
  2. 右上の「デベロッパーモード」をオンにする（スイッチをクリック）
  3. 「パッケージ化されていない拡張機能を読み込む」ボタンをクリック
  4. 解凍した reatex-helper-v1.2.1 フォルダを選択して「開く」
  5. 一覧に「REATEX 概要書ヘルパー」が表示されれば ✅ インストール完了
chrome://extensions/
3

ツールバーにアイコンを固定する 推奨

  1. アドレスバー右端の 🧩 パズルピースアイコンをクリック
  2. 「REATEX 概要書ヘルパー」の右にある 📌 ピンアイコンをクリック
  3. ツールバーに REATEX アイコンが常時表示されるようになります
✅ 固定しておくと、物件概要書ページでワンクリックで開けます。

📋使い方

基本的な流れ

① 物件概要書ページを開く
↓
② ツールバーの REATEX アイコンをクリック
↓
③ 右側にパネルが表示される
↓
④ 各タブの文章をコピーして使用

各タブの使い方

📧 メルマガタブ
  1. 「メルマガ」タブを選択
  2. 件名と本文（HTML）が自動生成される
  3. 「プレビュー」ボタンで実際のメールデザインを確認
  4. 「件名をコピー」「HTMLをコピー」ボタンで配信システムに貼り付け
ℹ️ 写真が概要書にあれば自動取得されます。ない場合は「写真準備中」と表示されます。
💬 業者LINEタブ
  1. 「業者LINE」タブを選択
  2. 業者向けLINE紹介文が自動生成される
  3. 短縮URLがある場合は入力欄に貼り付けて「短縮URLを反映して再生成」を押す
  4. 「LINE文面をコピー」ボタンでコピー
📲 Lステップタブ
  1. 「Lステップ」タブを選択
  2. 配信テキストが自動生成される
  3. 「Lステップ文をコピー」ボタンでコピー
🤖 AIチャットタブ

Claude API キーが必要です（社内担当者から入手）

  1. 「AIチャット」タブを選択し、初回のみ API キー（sk-ant-...）を入力して「保存」
  2. 「修正対象」でメルマガHTML / 業者LINE文 / Lステップ を選択
  3. 入力欄に修正指示を記入
    例：「挨拶文をカジュアルに変えて」「価格を太字にして」
  4. 「送信」または Ctrl+Enter で送信 → 文章が自動反映
🔒 APIキーは外部に漏洩しないよう厳重に管理してください。
🔄 パネル下部の「再抽出・再生成」ボタンを押すと、ページを読み直して全タブを再生成します。

❓よくある質問

パネルが開かない
ページを再読み込みしてからもう一度アイコンをクリックしてください。
写真が「写真準備中」になる
概要書ページに写真が掲載されていないか、読み込み中の可能性があります。ページを完全に読み込んでから「再抽出・再生成」ボタンを押してください。
物件番号が「(不明)」になる
対応していないURL形式のページです。物件番号は手動で確認してください。
APIキーを忘れた
AIチャットタブの「変更」ボタンを押すと再入力できます。
拡張機能をアップデートしたい
chrome://extensions/ から既存の拡張機能を削除し、新しいZIPを解凍して再インストールしてください。

⚠️注意事項

本ツールは Chrome 専用です
物件概要書ページを開いた状態でのみ動作します
生成された文章は必ず内容を確認してから使用してください
Claude APIキーは外部に漏洩しないよう管理してください
