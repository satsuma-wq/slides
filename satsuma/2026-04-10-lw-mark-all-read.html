LINE WORKS 全既読ブックマークレット

React 17/18 対応版（2026-04-10更新）

ステップ1：コードをコピーする

ボタンを押すとテキストが全選択されます。その後 Ctrl+C でコピーしてください。

ステップ2：ブックマークに登録する

  1. Chrome のブックマークバーで右クリック →「ページを追加」
  2. 名前：LW全既読
  3. URL 欄にコピーしたコードを貼り付け
  4. 保存
⚠️ Chrome はブックマーク登録時に先頭の javascript: を消すことがあります。
登録後に編集して URL が javascript:void( で始まっているか確認してください。
消えていた場合は先頭に javascript: を手動で追加してください。

ステップ3：使い方

  1. LINE WORKS を開き、トーク一覧が表示されている状態にする
  2. ブックマークバーの「LW全既読」をクリック
  3. 確認ダイアログが出るので「OK」を押す
  4. 未読チャットを順番に既読にします（処理中はタブのタイトルに進捗が表示されます）
